This site is built with [Astro](https://astro.build) and deployed to GitHub
Pages. A quick tour for anyone curious — or for future me.

## The stack

- **Astro** for static rendering with near-zero client JavaScript
- **Tailwind CSS** for styling
- **Content collections** for this blog and the projects list
- **PhotoSwipe** for the photography lightbox

## Why static

For a personal site, static output is hard to beat: it's fast, free to host,
and there's no server to maintain. Replace this post with your own thoughts.
